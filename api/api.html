TRANSACTION

The TRANSACTION function is a POST method. It has six parameters:
  1. transactDescription - Your description of the transaction
  2. amount - Amount must be greater than 0
  3. accountUserID - ClientID for XBank
  4. accountType (chequing or savings) - Account type will be using for the transaction
  5. accountID (chequingID or savingsID) - The accountID of the account will be using for the transaction
  6. transferAccountType (chequing or savings) - Account type of the account will be transfer the amount to
  7. transferAccountID - The accountID of the account will be transfering to
